# Tariq Gardens — A Life Beyond Living

A premium redesign of the [Tariq Gardens Housing Scheme](http://www.tgardens.com.pk/) website — LDA-approved master-planned community in Lahore, Pakistan.

## Preview Locally

```bash
python3 -m http.server 8787
```

Then open **http://localhost:8787/**.

## Features

- Cinematic full-screen overlay menu
- Bilingual support (English / اردو) with Noto Nastaliq Urdu
- Auto-rotating hero slideshow with parallax
- 32-image filterable gallery with lightbox
- Animated counters, scroll reveals, premium type (Cinzel, Cormorant Garamond, Inter)
- Fully responsive from 4K down to mobile
- Zero dependencies — static HTML/CSS/JS
